# Portfolio Website V2

This is a modern and dynamic portfolio website developed using HTML, CSS, and JavaScript. It features automatic integration with GitHub and Medium, allowing you to showcase your projects and blog posts seamlessly.

🌐 **Live Demo:** [efekurucay.com](https://efekurucay.com)

## 🚀 Features

- ✨ Modern and responsive design
- 🌓 Dark/Light mode support
- 🔄 Automatic GitHub projects integration
- 📝 Automatic Medium posts integration
- ⚙️ Content management through admin panel
- 🎨 Customizable interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub API
- Medium API

## 🏗️ Project Structure

```
Portfolio-Website-V2/

├── css/            # Stylesheet files
```

## 🚀 Installation

1. Clone the project
```bash
git clone https://github.com/efekurucay/Portfolio-Website-V2.git
```

2. Navigate to project directory
```bash
cd Portfolio-Website-V2
```

## 💡 Usage

1. **Admin Panel Access**
   - Navigate to `/admin`
   - Log in with your credentials
   - Manage your content through the dashboard

2. **GitHub Integration**
   - Projects are automatically fetched from your GitHub profile
   - They are displayed in the Projects section

3. **Medium Integration**
   - Blog posts are automatically pulled from your Medium account
   - They appear in the Blog section
